Здравствуйте.

Описание тестовой задачи находится в haulmont_TestTask.doc.

Для запуска приложения используйте распакуйте архив и запустите run.bat (можно запустить и по двойному щелчку).
В директории sources лежат:
1. исходные коды
2. используемые библиотеки (при запуске приложения они используются, поэтому для корректной работы не стоит их переносить)
3. скрипт создания ./sources/resources/scripts/createTables.sql (так же используется при запуске для создания базы данных)
